# NutriGram
Ai chatbot to recommend nutritious budget friendly meal plans

Twilio sandbox only allows registered users for now :( 

## Test the Bot Locally (No Twilio or WhatsApp Needed)
### Step-by-Step

```bash
# 1. Clone the repo
git clone https://https://github.com/RajwarManas/NutriGram
cd NutriGram

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the server
python app.py
